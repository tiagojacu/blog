Secuencia Fibonacci en JavaScript

La secuencia Fibonacci es una secuencia de números. Funciona de esta manera:

“Se parte de dos números iniciales, el 0 y el 1. Para sacar el número siguiente a estos dos debemos de sumar los dos últimos números (0 + 1). Tendremos como resultado el número 1, y quedaríamos con una lista de números tal que así ‘0, 1, 1’. Ahora, para sacar el número siguiente seguiremos los mismos pasos. Sumaremos los dos últimos números de la lista (1 + 1), y tendremos como resultado el número 2. Nuestra lista quedaría conformada por los números ‘0, 1, 1, 2’. Este proceso se repite las veces que sea necesario”.

En este artículo te explicaré como puedes codificar esta secuencia en JavaScript.

En primer lugar vamos a hacer uso de un dato de tipo array para almacenar solamente los dos últimos números de la secuencia Fibonacci. Con este array podremos hacer las operaciones de suma entre los dos únicos números que hayan en el array.

Luego, en una variable vamos a indicar la cantidad de números Fibonacci que queremos que aparezcan en pantalla. Esto lo hago porque hasta cierto punto (al escribir más de 1475 números) los números fibonacci más grandes empiezan a escribirse como un valor Infinity. Si lo deseas luego puedes hacer que estos números aparezcan completos.

Como la suma se va a hacer de manera cíclica, vamos a hacer uso del ciclo for. En él declararemos la variable i, que tendrá como valor inicial 0. Su condición será ‘mientras i sea menor a la variable fibonacciQuantity’. Por último, su expresión de incremento será sumar más uno a la variable i.

Dentro del ciclo for declararemos una variable result, que tendrá como valor el resultado al sumar los dos números que hayan en el array. Aquí hacemos uso de indices para indicar aquellos dos números.

Escribiremos esa variable result mediante el método document.write(). Para organizar los números fibonacci los separaremos con comas, como se ve en la imagen.

Por último, modificaremos nuestro array para cambiar los números que se encuentran en él. Primero eliminamos el primer número fibonacci que haya en el array mediante el método splice(). Luego introducimos el nuevo número fibonacci (el cual se encuentra en la variable result) al array mediante el método push().
